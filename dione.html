Dione: Multi-Environment Deployment

Dione enables the deployment of development environments in a variety of ecosystems, including on-premises infrastructure, cloud infrastructure, and container orchestration platforms like Kubernetes.

This flexibility allows developers to work within their required or preferred environments while leveraging the power of automation.

API Documentation

  • createEnvironment(environmentConfig): Creates a new development environment based on the provided configuration.
  • deployEnvironment(environmentID, deploymentOptions): Deploys the specified development environment with the given deployment options.
  • updateEnvironment(environmentID, updatedConfig): Updates the configuration of an existing development environment.
  • deleteEnvironment(environmentID): Deletes the specified development environment.

Examples

Example 1: Creating a development environment with AWS:

const environmentConfig = {
    name: 'my-development-environment',
    provider: 'AWS',
    region: 'us-east-1',
    instanceType: 't2.micro',
    subnetID: 'subnet-123456',
    securityGroups: ['sg-789012'],
    tags: { Environment: 'Development' },
    userData: '... base64 encoded user data ...',
    // Other configuration options...
};

dione.createEnvironment(environmentConfig);

Example 2: Deploying a development environment to Kubernetes:

const deploymentOptions = {
    targetCluster: 'my-kubernetes-cluster',
    namespace: 'development',
    deploymentStrategy: 'rolling-update',
    replicaCount: 3,
    serviceType: 'LoadBalancer',
    labels: { App: 'my-app' },
    // Other deployment options...
};

dione.deployEnvironment('environmentID', deploymentOptions);

Example 3: Updating environment configuration:

const updatedConfig = {
    instanceType: 't2.medium',
    securityGroups: ['sg-789012', 'sg-345678'],
    // Other updated configuration options...
};

dione.updateEnvironment('environmentID', updatedConfig);

Example 4: Deleting a development environment:

dione.deleteEnvironment('environmentID');
