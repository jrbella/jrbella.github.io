Saturn API

This documentation focuses on infrastructure deployment.

Endpoints

  • /deployments - Get all deployments
  • /deployments/{id} - Get deployment by ID
  • /deployments/create - Create a new deployment

Authentication

The Saturn API uses token-based authentication. You need to include a valid access token in the Authorization header of your requests.

Error Handling

The API returns appropriate HTTP status codes along with error messages in JSON format.

Example:

{
    "error": "Unauthorized",
    "message": "Invalid access token"
}
