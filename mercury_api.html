Mercury API

This is the API that interacts with all of the following submodules.

Getting API Token

To access the Mercury API, you need to obtain an access token by authenticating with your credentials.

Example using cURL:

curl -X POST \
  http://example.com/api/token \
  -H 'Content-Type: application/json' \
  -d '{
    "username": "your_username",
    "password": "your_password"
}'

Example Response:

{
    "access_token": "your_access_token",
    "token_type": "bearer"
}

Connecting to Saturn API

Once you have obtained the access token, you can use it to authenticate with the Saturn API.

Example using cURL:

curl -X GET \
  http://example.com/saturn-api/data \
  -H 'Authorization: Bearer your_access_token'

Connecting to Neptune API

Similarly, you can use the access token to authenticate with the Neptune API.

Example using cURL:

curl -X GET \
  http://example.com/neptune-api/data \
  -H 'Authorization: Bearer your_access_token'
