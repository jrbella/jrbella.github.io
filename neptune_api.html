Neptune API

This documentation focuses on using AI to optimize resources and cost.

Endpoints

  • /resource-optimization - Optimize resource usage
  • /cost-analysis - Analyze cost and suggest optimizations

Authentication

The Neptune API uses token-based authentication. You need to include a valid access token in the Authorization header of your requests.

Error Handling

The API returns appropriate HTTP status codes along with error messages in JSON format.

Example:

{
    "error": "Unauthorized",
    "message": "Invalid access token"
}
